## Tech Stack

### Backend
- Java 17, Spring Boot 3.x  
- Spring Data JPA, Hibernate  
- QueryDSL  
- Spring Security, OAuth2  
- 이벤트 기반 처리(After-Commit), Scheduler, Async

### Database & Cache
- MySQL, MariaDB  
- PostgreSQL  
- Redis, Redisson(분산 락, 캐싱)

### Infra & DevOps
- Kafka  
- Docker, Docker Compose  
- AWS (EC2, RDS, S3, ALB, IAM)  
- GitHub Actions(CI/CD)

### 기타
- REST API 설계  
- OpenAI GPT / TTS / STT API 연동 경험  
- 로그 기반 구조 분석 및 리팩토링 경험
